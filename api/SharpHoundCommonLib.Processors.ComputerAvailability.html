IsComputerAvailable(String, String, String, String)

Checks if a computer is available for SharpHound enumeration using the following criteria: The "operatingsystem" LDAP attribute must contain the string "Windows" The "pwdlastset" LDAP attribute must be within 60 days of the current date by default. Port 445 must be open to allow API calls to succeed
